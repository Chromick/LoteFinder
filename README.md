# LoteFinder Itaipuaçu

Web app instalável para apoio em manutenção de fibra em Itaipuaçu.

## Uso

1. Digite o nome atual da rua, como aparece no Google Maps.
2. Digite a quadra.
3. Abra no Google Maps ou Waze.
4. Quando estiver na quadra correta, salve o GPS atual.

Depois que o GPS de uma rua + quadra estiver salvo, o app passa a abrir direto no ponto confirmado.

## Instalação no telefone

Publique este repositório com GitHub Pages e abra o link HTTPS no Chrome do telefone.

No Chrome:

1. Abra o menu.
2. Toque em `Adicionar à tela inicial` ou `Instalar app`.

## Observação

O PDF original tem nomes antigos ou numéricos para várias ruas. Por isso, a busca usa o nome atual digitado no campo de rua e consulta o Maps.
